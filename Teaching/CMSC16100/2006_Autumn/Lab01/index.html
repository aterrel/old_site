Lab 1 - Intro to Dr. Scheme

The Development Environment

For this lab course we will be using the Dr. Scheme Integrated Development Environment (IDE). This environment is nice for basic programming and will provide you with some experience that will allow you to quickly learn other IDE's. For those of you who have never programmed, a development environment is a computer environment in which you program. It is usually dependent on your OS, language compiler/interpreter, text editor and other such things. An IDE is a development environment that knows how to compile and run programs on your project, it sometimes bolds keywords and such as well.

For this lab you are to first complete the Dr. Scheme Tutorial. Then you will make some example programs and save them to turn in using the maclab dropbox..

Learning Dr. Scheme

  • Go to the Dr. Scheme Tutorial off the maclab website.
  • Read through the tutorial, we will actually be using the dropbox instead of the handin script. For further info on the drop box, look here
  • Play around with the interaction and definition windows to get use to things. If you have questions just ask.

Writing you own program

You will be writing three functions today. The notorious "Hello World", a program to return a person's name and salutation, and a function to print out the factorial of an integer. Please remember to comment your code in the manner described in class: 

  ;; Purpose: The purpose of the program
  ;; Contract: type in -> type out
  ;; Example: (function input) -> output

Also at the beginning of any file you turn in please add the following comments to aid the graders. 

  ;; Name: My Name
  ;; Date: today's date
  ;; Class: CMSC 16100-1

Hello World

To print a string in Scheme, you use the display command, that is (display string) but alas this doesn't work in Beginner Student Mode of Dr. Scheme. To print out "Hello World!" bind a variable to the string and call it at the end of your definitions window.

Conditional Hello

In class you learned how to use the cond statement. Using the cond statement write a function that will take as input a name and sex. The function should return Hello Mr or Ms (depending on sex) name, use the string-append and equal? commands.

Factorial

To jump ahead a bit you will write a function that returns the factorial of a number. To do this have a function take a number as input, if the number is zero return one, else return the number times the factorial of one minus the number. You can do this by calling the same function with the number minus one as its argument.

Hand in

For this lab you only need to hand in the definitions window, but I would appreciate it if you left test cases in your definitions window. That is at the bottom of the window add the function calls such as: 

  hello_world
  (hello "Andy" "m")
  (fact 3)

Save your file as (your-cnet-id)-Lab01.scm and use the dropbox to turn it in. If you ever need to make changes after you have already dropped something off, you can turn in a newer version and mark it by saving as (original_name)-version-x.scm, I will only look at the latest version.
